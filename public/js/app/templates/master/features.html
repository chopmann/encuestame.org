Features

TweetPoll

Simple and Fast. It is how you can create and share your micro-surveys using the most extensive network to share, engaging your followers and voting in real time.

Poll

Surveys of a single question with multiple distribution options. Real-time results.

Survey

Create and manage your online surveys easily and efficiently. It uses multiple types of responses. Types of responses can include multimedia (images, videos). Ability to georeference the place of capture / collection of survey data, privacy options and many more options. In development

Public Profile

It is the user's personal page. Channel has RSS / ATOM for distribution. Customize the privacy of the content published, the profile displays the record of his last publications or actions.

Customize your Profile

Update your personal data, upload a new profile image or use the Gravatar source.

Graphs

Encuestame have differents type of charts to display the survey stats, pie charts, Bar charts, Lines etc...

Easy 7 steps

Easy and intuitive Wizard. Follow the installation steps where you can customize your configuration database, create demonstrations data in a single 7-step guide.

Multi-Database compatibility

Encuestame use Hibernate with highly support the most of popular databases, MySQL, PostgesSQL, MSSQL Databases, Oracle and more.

Mobile Support

Surf on the web mobile it's very easy, Encuestame have a powerful user interface to display survey stats, details and other features on mobile devices.

Instant Search

Predicting your search and showing results before you finish typing.

OAuth

Encuestame has OAuth capabilities, use to connect to most modern social networks on versions 1 and 2.

Filters

All managed interfaces has differents ways to filter the data, find easily surveys, filter by social networks, by dates and another type of filtres are available.

Connect

Easy to connect your favorites social networks as Facebook, Twitter, LinkedIn, identi.ca, etc.

Easy Sign Up

If you have a Facebook account, use the easy sign up to create a new account, only in one step

Masive Distribution

Publish on multiple social networks, select a list of social accounts to make a massive distribution in only one click.

Quick Manage

Organize your survey it's very easy, save your favorites and group on folders.

Security Manage

Create users, groups, assing permissions it's very easy, organize it's now a fun task. In development

REST API

Encuestame RESTful services is based on the interaction with the interface is 90% via REST, these services can be used by third party applications such as mobile applications (Android, IOS) or other applications consuming REST services.
