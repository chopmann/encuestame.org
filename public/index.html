About Us

History

Encuestame it's a free open source project created in 2008 in Nicaragua and Spain , now our headquarters are in Austria. The project start as PHP project, migrated later to Java, we decided create RESTFul API to integrate the new User Interface with three party applications. On 2011 we recieved a Atlassian Open Source Licenses, this made more mature to the project.

RoadMap

This project is alive, we are working every day to offer to you a more stable product, all related with the Roadmap is available in this page.

Encuestame doesn't have incomes and is non profit, it's a free project and only accept donations to maintenance the infrastructure. The best donation for us is your own code.
